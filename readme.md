## Installation:

### Install npm modules:
```bash
$ npm i
```

### Run project:
```bash
$ npm start
```

### Build app:
```bash
$ npm run build
```

### Check types in run time
Open .babelrc.

```
"optInOnly": false, // check types in run time
"optInOnly": true, // DO NOT check types in run time
```
